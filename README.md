<!-- ## Introduction -->

## Requirements

- [yarn](https://yarnpkg.com/) for package management

## Installation

First, install [yarn](https://yarnpkg.com/).

From the `./mtaa-frontend/` directory, you can install all dependencies with:

```bash
yarn install
```

## Running the Project

You can run the project with:

```bash
yarn run start
```
